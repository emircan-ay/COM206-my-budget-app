Wallet App

This project is a simple expense tracking application built with Flutter.  
Users can add expenses using a price and description, view them in a list, and delete them.

---

 Features

- Add expenses (price + description)
- Display expenses in a list
- Delete expenses
- Modern and clean UI
- Data input using TextField
- Dynamic UI updates using `setState`

---

 Technologies Used

- Flutter
- Dart
- StatefulWidget
- ListView.builder
- TextEditingController

---

 UI Overview

 First Page

- Body:
  - Text
  - ElevatedButton (navigates to the second page)

 Second Page

- AppBar: "Add Expense"

  Top Section:

- Price input field
- Description input field
- Add button

 Bottom Section:

- List of expenses (Card + ListTile)
- Delete button
